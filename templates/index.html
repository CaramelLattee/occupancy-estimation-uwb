About This Project

This dashboard provides a local web interface for the FYP occupancy estimation system using two Qorvo DWM3001CDK UWB radar boards. The Live Dashboard streams real-time ranging data from both boards and provides an instant occupancy estimate based on RSSI thresholds. The Data Viewer allows browsing and visualising all collected HDF5 sessions with signal distribution charts. The ML Results page will display classifier performance once training is complete.
