Key insights
Label 0 (Empty room) — 100% F1. The empty room produces a uniquely stable signal with tight distance IQR (183–188 cm) and RSSI spread of only ±2 dBm. The classifier correctly identifies every empty room instance with zero errors.
Main confusion: Label 2 vs Label 3. 5 out of 16 Label 3 windows are misclassified as Label 2. This is physically expected — two and three people produce similar RSSI attenuation patterns. CIR-based features (micro-Doppler) should resolve this confusion.
RSSI dominates feature importance. All top 10 features are RSSI-based (mean, std, min, IQR, percentiles) from both boards. Distance features contribute less — confirming that signal strength variation is the primary discriminative cue for occupancy estimation.
83.0% macro F1 exceeds initial estimate of 78–88%. The strong progressive RSSI trend observed in preliminary analysis (−62 → −64 → −65 dBm) translates directly into high classification accuracy for the DIAG baseline, above the lower end of the literature estimate.
